# 每日行情营销内容生成

> 用户通过 `/generate-marketing` 或者说"帮我生成今日行情文案"触发此工作流。

## 背景
- 本工作流直接由 AI 助手读取本地数据库、分析趋势，并生成类“小鱼判断”风格的深度硬件日报。
- 替代了之前简陋的统计报表，增加了细分品类（如 DDR4/DDR5，1TB/2TB）的对比和历史极值追踪。

## 工作流程

### 1. 拉取细粒度行情数据
// turbo
运行最新编写的数据拉取脚本，自动获取今日各板块核心数据（含历史异常值、DDR4/DDR5分类）：
```bash
python3 /Users/mac/new/server_py/scripts/generate_daily_report_data.py
```

### 2. 生成特色四端营稿 (严格参照“显卡日报”模版)
根据上述命令吐出的 JSON 数据（`change_7d`, `anomalies`），**直接由 AI 生成**以下四端文案。

**核心模版结构（必须严格遵循此流程）：**
1. **开场叙述与大盘定调**：“欢迎来到显卡日报，今天[核心变动硬件]继续回调...”
2. **内存细分深度分析**：
   - 必须报出 DDR4 和 DDR5 的具体均价和 7 天降幅。
   - 必须引用 `anomalies` 里的数据：“[具体型号] 降幅达到 [XX] 元，目前来到了 [XX] 元。”
3. **主理人判断（必须包含）**：“**小鱼判断**，因为[结合当前大环境编撰理由]，但长远来看...”
4. **硬盘与 CPU 对比**：盘点 1TB、2TB 的差异，对比 Intel 与 AMD 涨跌幅度。指出版单里涨跌最离谱的型号（如 "属于历史新低，建议加 X 元换 Y"）。
5. **单栏显卡情况**：简述 GPU 波动。
6. **固定结尾**：“今天的日报就到这里，每晚准时更新，喜欢的可以点关注，”

#### 多端适配：
- **公众号 / 长文**：完全按以上模板展开，字数 800 左右。
- **小红书**：配 Emoji，核心提取“小鱼判断”和那几个大跳水的单品。
- **朋友圈**：100字，只报跌幅最大的两款硬件。
- **短视频**：开局念大盘涨跌件数，然后直接念“小鱼判断”。

### 3. 生成可交付的本地大盘 HTML 战报
使用 `write_to_file` 工具在 `/Users/mac/Downloads/今日硬件行情_战报_<日期>.html` 下生成一份 Tailwind + ECharts 单页文件。
HTML 中需包含：
- 数据表格。
- 四端文案区块，需提供【一键复制】按钮。
- **注意**：图表必须包含数据的涨跌幅可视化。

### 4. 给用户发送通知
通过 `notify_user` 简要预告今日核心数据，并提示用户去 Downloads 文件夹打开 HTML 预览文案效果。
