# Game Icons attribution

The SVG icons in this folder are derived from the
[`game-icons/icons`](https://github.com/game-icons/icons) repository.

They are used under Creative Commons Attribution 3.0 unless the upstream
repository marks an individual contributor's work as CC0.

- Icons by Delapouite: `cpu`, `ram`, `gpu`, `power`, `case`, `fan`, `monitor`,
  `mouse`, and `keyboard`.
- Icons by Lorc: `cooling`, `mainboard`, `disk`, and `accessory`.

Local changes: the solid background path was removed so the icons can be
displayed on themed category tiles. The icon silhouettes are otherwise kept
unchanged.

Source repository: https://github.com/game-icons/icons
