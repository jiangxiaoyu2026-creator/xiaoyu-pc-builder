# Mario game theme assets

The embedded sprite frames in this directory were sourced from
[Super-Mario-Phaser](https://github.com/pablogozalvez/Super-Mario-Phaser).

Source files:

- `assets/entities/mario-grown.png`
- `assets/entities/koopa.png`
- `assets/scenery/pipe1.png`
- `assets/blocks/overworld/block.png`
- `assets/collectibles/super-mushroom.png`
- `assets/scenery/castle.png`
- `assets/scenery/overworld/cloud1.png`
- `assets/scenery/overworld/cloud2.png`
- `assets/scenery/overworld/floorbricks.png`
- `assets/entities/overworld/goomba.png`

`mario-pointer.svg` is an original theme illustration created for this project
and does not copy sprite pixels from the source repository. The animated wings
in `paratroopa.svg` are an original pixel treatment around the sourced Koopa frame.

The following supporting silhouettes were derived from
[`game-icons/icons`](https://github.com/game-icons/icons), then recolored and
rasterized at a low resolution for this pixel theme:

- Delapouite: `carnivorous-plant`, `chicken`, `chimney`, and `spiky-pit`.
- Lorc: `sparrow`.
